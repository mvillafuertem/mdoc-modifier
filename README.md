### Troubleshooting :scream:

#### Can not find Graphviz

* Install Graphviz

more information see http://plantuml.com/es/graphviz-dot

* Add this line to diagram
    
```
!pragma graphviz_dot jdot
```

more information see http://plantuml.com/es/smetana02

```plantuml

@startuml

!pragma graphviz_dot jdot
class Foo1

Foo1 --> Foo2
Foo1 --> Foo3
Foo1 ---> Foo4 : test 4
Foo1 ----> Foo5 : test 5

@enduml

```
